PROJECT DEPENDENCIES AND THEIR PURPOSE

1. pandas

Purpose:
Used for data manipulation and analysis.

How it was used in this project:
- Loading customer datasets
- Cleaning and organising data
- Handling customer records
- Creating analysis tables
- Preparing data for machine learning


2. numpy

Purpose:
Used for numerical calculations and array operations.

How it was used in this project:
- Handling numerical data
- Supporting machine learning calculations
- Managing prediction outputs


3. scikit-learn

Purpose:
A machine learning library used for building predictive models.

How it was used in this project:
- Splitting data into training and testing sets
- Encoding categorical variables
- Scaling numerical features
- Training the Logistic Regression model
- Evaluating model performance
- Generating churn predictions


4. streamlit

Purpose:
Used to create interactive web dashboards using Python.

How it was used in this project:
- Building the customer churn dashboard
- Creating interactive pages
- Displaying customer metrics
- Creating prediction interfaces
- Presenting machine learning results to users


5. plotly

Purpose:
Used for creating interactive data visualisations.

How it was used in this project:
- Creating churn charts
- Displaying customer behaviour patterns
- Building interactive dashboard graphs


6. matplotlib

Purpose:
A Python visualisation library used for creating charts.

How it was used in this project:
- Creating statistical visualisations
- Supporting the satisfaction score analysis
- Generating box plots


7. seaborn

Purpose:
A visualisation library built on top of matplotlib.

How it was used in this project:
- Creating the customer satisfaction box plot
- Improving statistical chart presentation
- Analysing relationships between customer variables


8. joblib==1.4.2

Purpose:
Used for saving and loading machine learning objects.

How it was used in this project:
- Saving the trained Logistic Regression model
- Saving the feature scaler
- Saving feature names
- Loading the model into the Streamlit dashboard
